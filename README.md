# NEAP Client-Side

This is the client-side of our NEAP App. It consumes data from our server using Angular.

## Development

1. Create a new component
1. Set up a service
1. Controller
  - Access the service
1. Service
  - AJAX
1. Add `access-control` header check to server
1. Controller
  - Update the view

## Deployment

`$ git subtree push --prefix src origin gh-pages`
